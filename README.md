# Gamepad Store — hero + каталог

Тестовая вёрстка одного блока интернет-магазина геймпадов: полноэкранный hero и ряд из четырёх карточек товара. Чистый HTML + CSS, без фреймворков, бутстрапа, Tailwind и UI-китов — всё руками.

Стиль — светлый editorial-минимализм: белый фон, индиго-акцент (`#4F46E5`), крупная типографика, много воздуха. Hero на весь экран с фоновым фото, которое бесшовно растворяется в белом каталоге.

## Стек

- HTML5 (семантика `header` / `section` / `article`)
- CSS3 (custom properties, grid, `clamp()`, `aspect-ratio`)
- Шрифт [Manrope](https://fonts.google.com/specimen/Manrope) (Google Fonts)
- Без сборки и зависимостей

## Структура

```
gamepad-store/
├── index.html      # разметка
├── style.css       # стили (токены, layout, ховеры, адаптив)
├── images/         # локальные фото геймпадов
├── vercel.json     # конфиг статического деплоя
└── README.md
```

## Локальный запуск

Открыть `index.html` в браузере или поднять статик-сервер:

```bash
python3 -m http.server 8910
# http://127.0.0.1:8910
```

## Деплой на Vercel

Проект — чистая статика, Vercel подхватывает его без настроек (framework preset: *Other*). `vercel.json` лишь включает `cleanUrls` и кэширование `/images`.

```bash
vercel
```

или импорт репозитория в дашборде Vercel.

## Адаптив

- Десктоп — полноэкранный hero, каталог `auto-fit` (до 4 колонок).
- Точки перелома: `768px` и `480px` (кнопки hero на всю ширину, поджатый ритм, одна колонка карточек).
- `100svh` для hero, `prefers-reduced-motion` отключает анимации.

## Фото

Изображения — с [Unsplash](https://unsplash.com) (свободная лицензия), скачаны локально в `images/`. Если файл не загрузится, карточка показывает аккуратную графитовую заглушку.
